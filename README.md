# Calender With Add Event (Costwise Assignment)

## Run Locally

Clone the project

```bash
  git clone https://github.com/Raj-Stark/costwise_assignment.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Features

- 12 months Calender
- Can Add Events
- Can Delete Events
- Can Edit Events
- Can Change Months Using Slider
